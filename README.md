# Float Finder — Electron App

## Setup

Run once to install Electron:
```
npm install
```

## Start (Dev)
```
npm start
```

## Build Windows EXE

### Installer (.exe setup):
```
npm run build
```

### Portable (.exe no-install):
```
npm run build:portable
```

Output goes to `dist/` folder.

## Notes
- Requires Node.js ≥ 18
- Icon: Place a 256x256 `.ico` file at `assets/icon.ico`
